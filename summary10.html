this

this是JavaScript语言的一个关键字。它代表函数运行时，自动生成的一个内部对象， 只能在函数内部使用。 

	1、 全局指向 Object Window
	var x = 2;  //如果去掉，则结果为undefined
	function test(){
		var x = 1; //如果把var去掉，则下面的返回值为1
		console.log(this.x); // 2
	}
	test();
	
	
	
	2、作为对象方法的调用，这时this就指这个上级对象。
	function test(){
		x = 2;
		console.log(this.x);
	}
		var o = {};
		o.x = 1;
		o.m = test;
		o.m(); //返回值是 1，因为这时函数test给了对象o中的m,
				调用o.m()时，this指的就是o

	3、作为构造函数调用，所谓构造函数，就是通过这个函数生成一个
	新对象(object)。这时，this就是这个新对象。
	function test(){
				
		this.x = 1;
	}
	var o = new test();
	console.log(o.x); //返回值为1
	
	
	
	4、apply调用
	apply()是函数对象的一个方法，它的作用是改变函数的调用对象，
	它的第一个参数表示改变后的调用这个函数的对象。因此，this指
	的就是这第一个参数。
	var x = 2;
	function test(){
		console.log(this.x);
	}
	var o = {};
	o.x = 1;
	o.m = test;
	o.m.apply(); //返回值为2
	
	apply()的参数为空时，默认调用全局对象。
	如果最后一行代码改为o.m.apply(o);时，返回值为 1 
	这时的this指的就是对象o

prototype和_proto_

	prototype是函数的一个属性（每个函数都有一个prototype属性）,这个属性是
	一个指针，指向一个对象。它是显示修改对象的原型的属性。
	_proto_是一个对象拥有的内置属性（请注意：prototype是函数的内置属性，
	_proto_是对象的内置属性），是JS内部使用寻找原型链的属性。
	
	
	new的过程：
	var Person = function(){};
	var p = new Person();
	
	上面过程可以拆分为一下三步：
	（1）、var p = {};
	（2）、p._proto_ = Person.prototype;
	（3）、Person.call(p);//初始化p
	
	实例：
	var Person = function(){};
	Person.prototype.sayName = function(){
		alert("Hello World!");
	};
	
	Person.prototype.age = 25;
	var p = new Person();
	p.sayName();
	
	p是一个引用指向Person的对象。我们在Person的原型上定义了一个
	sayName方法和age属性，当我们执行p.age时，会先在this的内部查
	找（也就是构造函数内部），如果没有找到然后再沿着原型链向上追溯。
