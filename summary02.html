JavaScript编码规范

通用命名规范

  • 所有变量必须是有意义的英文，严厉禁止拼音；
  • 变量命名采用小驼峰法（第一个单词首字母小写，其余单词首字母大写）；
  • 变量允许使用公认英文缩写，例如nav；
  • 常量必须所有单词大写，并且每个单词间加下划线；
  • 类命名必须是大驼峰法（所有单词第一个字母均大写）；
  • 私有类的变量属性成员，建议使用混合式命名，并在前面加上下下划线；
  • “on”只能用作事件的命名；
  • 所有全局变量必须初始化；
  • 保留字以及特有的dom属性不作为变量名。

变量命名规范：类型前缀+有意义的单词

  • 字符串：sXXX，如：sNname；
  • 数字：nXXX，如：nPage；
  • 逻辑：bXXX，如：bChecked；
  • 数组：aXXX，如：aList；
  • 正则：rXXX，如：rDomain；
  • 函数：fXXX，如：Getlist;
  • DOM节点：dXX，如：dDiv；
  • 其他类型：oXXX，如：oButton；
  • 特殊简写：小范围作用域临时变量，如函数内部变量或参数：o(Object)、e(Element)等；
  • 循环变量：i、j、k以此类推。

运算符

"="号用于赋值；
+"号用于文本值或字符串变量加起来（连接起来）。

		var a = +[];
		console.log(a);
		结果：0
	
		var b = +[];
		console.log(b);
		结果：NaN

JavaScript的数据类型

  • 数字number;(浮点数不能用"=="进行判断，因为它是近似数，例：10个0.1相加的结果并不是1)
  • 字符串string;
  • 布尔型boolean(结果只有true或false)；
  • 数组Array()；
  • 对象Object()；
  • Null空

函数

  • 函数声明共有三种方式：

  • 函数声明: function foo(){}
  • 表达式声明: var foo = function(){}
  • 匿名声明: function(){}

关于函数的调用，由于函数是“一等公民”，所以在调用时若其置于最上面也是可以调用到的。 还有调用的执行顺序是由上至下的。但是当同时调用表达式声明的函数和函数声明的函数时，表达 式函数的优先级大于后者。 

		var foo = function(){
			alert(0);
		}
		function foo(){
			alert(111);
		}
		foo();
					
		结果显示：弹窗为0。
	
匿名函数自调用：(function(){alert(0);})();

匿名函数只能在其内部作用，外部是无法调用的。

eval()函数

eval() 函数可计算某个字符串，并执行其中的的 JavaScript 代码。

实例如下

		eval("x=10;y=20;document.write(x*y)");
	
		其结果为：200
