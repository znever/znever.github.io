1. alter()属于window对象的方法

2. JavaScript函数默认返回的值是undefined。

3. setInterval在执行完一次代码之后，经过了那个固定的时间间隔， 它还会自动重复执行代码，而setTimeout只执行一次那段代码。 它们的指向是window 

	4. var a = 1;
	function test(){
		a++;
		alert(a);
		var a = 2;
		alert(a);
	}
	test();
	返回的结果分别是：NaN,2
	因为在test函数里声明了一个局部变量a;
	但是alert()时在其上面且a++,所以返回NaN。
	第二个alert()返回时是函数里的局部变量a = 2.
