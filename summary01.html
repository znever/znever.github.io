GitHub

1.进入GitHub官网，如果没有账号，首先需要注册（sign up）一个，注册成功之后便可登录（sign in）；
2.点击右上角的小图标找到settings，然后验证邮箱；
3.点击右上角的图标加号选择New repository新建一个项目。
4.在Repository name处输入项目名，勾选下面的Public，点击Create repository。
5.点击settings,再创建一个仓库名称， 点击save project。
6.点击settings，然后点击Launch automatic page generator，点击Continue to layouts。
7.在master中找到主文件，可以修改index.html文件或者上传新的文件便可搭载完成。

Hbuilder

官网下载一个Hbuilder软件（http://www.dcloud.io/），安装并打开Hbuilder软件。 首先新建一个web项目，Ctrl+N,选择web项目，输入项目名称，也可以新建其他文件。 此软件可以快速生成代码，例如标签后可以乘以数目（div*n）就可以生成n个div。 可以从外部导入项目或文件，也可以修改软件工具的皮肤。

SVN

这个软件工具最大的有点就是可以同时用于多个人共同开发一个项目， 并且在最后项目提交的时候可以智能的识别多个人之间修改统一代码之间出现的差异。
